grading submissions, and managing student training.

⚡ QUICK START (3 Steps):

Step 1: Enable GPT-5
   Open Microsoft Copilot or ChatGPT
   Go to Settings → Enable GPT-5
   (Required for medical accuracy)

Step 2: Load Core File
   Drag and drop EMSTrainer_Core.txt into the chat
   Wait for: "[Core loaded successfully]"

Step 3: Load Instructor Interface
   Drag and drop EMSTrainer_Instructor_Interface.txt
   Wait for confirmation message

✅ You're ready! Say: "Create a cardiac arrest scenario"

═══════════════════════════════════════════════════════════════════

🎯 WHAT YOU CAN DO:

🎬 Create Scenarios
   - Natural language: "Create trauma scenario for paramedic students"
   - Customize difficulty, patient details, complications
   - AI generates complete JSON scenario file
   - Deploy via email or file distribution

📝 Generate Tests
   - Topic-specific: "Generate test on ALS airway management"
   - Skill-level targeting: "BLS test for EMT students"
   - Question count control, mixed topics
   - Student + instructor versions (with answers)

📚 Create Study Guides
   - "Create study guide on pediatric respiratory emergencies"
   - Customizable depth and focus
   - Includes practice questions
   - Ready for student distribution

✅ Auto-Grade Submissions
   - Paste student submission JSON
   - Instant scoring with detailed feedback
   - Strengths, weaknesses, timing analysis
   - Recommended improvements

📊 Class Summaries
   - Aggregate multiple student submissions
   - Average scores by category
   - Common mistakes identified
   - Performance trends

💡 PRO TIPS:

✅ Start with Examples
   Load examples/scenario_*.json and modify them
   Faster than creating from scratch

✅ Match Difficulty to Students
   Easy = New students learning basics
   Standard = NREMT prep, most students
   Hard = Experienced providers
   Monica = Reality checks only!

✅ Focus on 2-3 Learning Objectives
   Don't overwhelm with too many concepts
   Clearer learning outcomes

✅ Review Auto-Grades
   AI grading is good but review before releasing
   Add personal instructor notes

✅ Track Trends
   Use class summaries to identify common weaknesses
   Adjust teaching focus accordingly

🔄 TYPICAL WORKFLOW:

1. Create Scenario
   "Create respiratory distress scenario, Standard difficulty"

2. Review & Customize
   AI generates JSON, you can modify via chat

3. Deploy to Students
   "Format for email distribution to: [student names]"

4. Collect Submissions
   Students return JSON files

5. Grade Submissions
   Paste JSON: "Grade this submission"

6. Generate Summary
   "Create class summary for all submissions"

7. Provide Feedback
   Review, add notes, return to students
