🚑 Welcome to EMSTrainer!

This package contains everything you need to start training with
realistic EMS scenarios, practice questions, and study guides.

═══════════════════════════════════════════════════════════════════

📦 PACKAGE CONTENTS:

1. EMSTrainer_Core.txt              - Foundation (Test/Study modes)
2. EMSTrainer_Student_Interface.txt - Student interface
3. Student_Quick_Start_Guide.md     - Getting started guide
4. WHATS_NEW_v1.6.1.md             - Release notes

═══════════════════════════════════════════════════════════════════

⚡ QUICK START (3 Steps):

Step 1: Enable GPT-5
   Open Microsoft Copilot or ChatGPT
   Go to Settings → Enable GPT-5
   (Required for medical accuracy)

Step 2: Load Core File
   Drag and drop EMSTrainer_Core.txt into the chat
   Wait for: "[Core loaded successfully]"

Step 3: Load Student Interface
   Drag and drop EMSTrainer_Student_Interface.txt
   Wait for confirmation message

✅ You're ready! Say: "I want to practice scenarios"

═══════════════════════════════════════════════════════════════════

🎯 WHAT YOU CAN DO:

🎬 Practice Scenarios
   - Realistic patient encounters
   - Four difficulty levels (Easy → Monica)
   - Immediate feedback and debriefs
   - Say: "Run a cardiac arrest scenario in Standard mode"

📝 Practice Questions
   - Targeted questions on your weak areas
   - Upload PlatinumPlanner test results
   - NREMT-style questions with explanations
   - Say: "Generate 10 questions on cardiology"

📚 Study Guides
   - AI-generated study materials
   - Focused on specific topics
   - Includes examples and mnemonics
   - Say: "Create a study guide on airway management"

═══════════════════════════════════════════════════════════════════

💡 PRO TIPS:

✅ Use TEMPORARY CHAT MODE
   Prevents AI hallucinations during long scenarios
   Start fresh session for each scenario

✅ Upload Your Test Results
   See Student_Quick_Start_Guide.md for PlatinumPlanner instructions
   Get personalized training on weak areas

✅ Start with Easy Mode
   Build confidence before attempting harder scenarios
   Progress: Easy → Standard → Hard → Monica

✅ Read the Debriefs
   That's where the real learning happens
   Review what you did well AND what to improve

═══════════════════════════════════════════════════════════════════

🎓 DIFFICULTY LEVELS:

Easy Mode      - Learning-focused, no penalties, supportive
Standard Mode  - NREMT-level, realistic, balanced
Hard Mode      - Strict timing, equipment failures, challenging
Monica Mode    - EXTREME (experienced providers only!)
